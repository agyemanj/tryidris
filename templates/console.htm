Using Idris

Idris is a language for practical, dependently typed, functional programming. The Idris Tutorial is the best resource for becoming familiar with the language. You can use the REPL above to play around without installing the compiler on your computer. Give these examples a go:

"Hello world"
reverse (the (Vect 3 _) [1, 2, 3])
the (n ** Vect (S n) Int) (3 ** [1, 2, 3, 4])

You can also compile Idris to JavaScript from this service.
